## Eclipse Che

Those future project are built around my way of viewing Eclipse Che and how i work with it. It's not, at least now, how it's commonly viewed.

My view of Eclipse Che include the usage of [mise](https://mise.jdx.dev/) to describe the package needed in pair with [Weebo DotFile Che](https://github.com/batleforc/weebo-dotfiles-che) who is the base of all my workspace.
